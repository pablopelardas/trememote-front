## Tech Stack

**Client:** React, Redux Toolkit, Tailwind

## Features

- Show user Posts.
- Show posts by User
- Create and Edit posts

## Run Locally

Clone the project

```bash
  git clone https://github.com/pablopelardas/trememote-front
```

Go to the project directory

```bash
  cd trememote-front
```

Install dependencies in both folders (API and CLIENT)

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
